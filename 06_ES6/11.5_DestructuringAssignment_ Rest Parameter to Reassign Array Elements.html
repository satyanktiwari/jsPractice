Use Destructuring Assignment with the Rest Parameter to Reassign Array Elements

const [a, b, ...newArray] =
[1, 2, 3, 4, 5,6,7];
console.log(a, b);
console.log(newArray);
The console should display the values 1, 2 and [3, 4, 5,6,7].
