Slice

                      
                        slice() copies or extracts 
                        a given number of elements to a new array, 
                        leaving the array it is 
                        called upon untouched.

Example

                      
                        let weatherConditions = ['rain', 'snow', 'sleet', 'hail', 'clear'];
                        let todaysWeather = weatherConditions.slice(1, 3);
                        would create new array
                        todaysWeather =['snow', 'sleet']
                        and weatherConditions remains untouched.
